# Flask app

Using Flask to build a Restful API Server with Swagger document.


## Installation

Install with pip:

```
$ cd belvo-app-belvo-api

$ pip install -r requirements.txt
```

# Flask
### Run app for develop 


```
$ python main.py
```
In flask, Default port is `8001`

# Docker
### Run app for develop with docker

```
$ mv docker-compose.yml ../

$ docker compose up -d
```
